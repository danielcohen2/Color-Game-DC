What is RGB?

The RGB color model is an additive color model in which red, green, and blue light are added together in various ways to reproduce a broad array of colors. Each parameter (red, green, and blue) defines the intensity of the color as an integer between 0 and 255. For example, rgb(0, 0, 255) is rendered as blue, because the blue parameter is set to its highest value (255) and the others are set to 0.

How to Play?

At the beginning of each round, you will have 3 seconds to study the RGB color at the top of the page. Then, six squares of different colors will appear and you will have to guess the correct square that indicates the RGB color. Each round has a 10 second countdown so the quicker you guess, the more points you will receive. However, be careful, if you guess wrong (or don't decide before the timer runs out), you lose a life!
